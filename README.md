# multimodal
Public Transit app for UK and Germany.

This repo is a tracking repo. The original and cardinal:

https://openrepos.net/content/anarchyintheuk/multimodal

Routing and departure boards for National Rail and Transport for London (TfL) services:
* Trains
* Buses
* Trams
* Tube
* Overground
* TfL rail
* DLR

Departure boards for Metrolink and bus services in Manchester.
* Trams
* Buses

Routing and departure boards for Deutsche Bahn and S-Bahn services in Germany.
* National trains
* International trains
* S-Bahn: Berlin, Cologne, Hamburg, Hannover, Munich, Rhein-Main, Rhein-Neckar, Rhein-Ruhr, Stuttgart plus a few more.

For comments please check out the Sailfish OS Forum

For the map to be displayed Mapbox GL Native bindings for Qt QML and Mapbox GL Native library  package has to be installed.
